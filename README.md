# odin-battleship

# Description
This is a demo web-based version of the board game Battleship. 

# Usage
Play against the computer by selecting P1, or against another player (on the same device) by selecting P2.
To play against another player, make your move and pass the device between each player. Make sure the other player can't see you setting up your ships!
Click and drag the ships to put them in place during the setup phase. Once both players are ready, attack by clicking the desired location on the front facing board.

# Skills 
- Interactive web app powered by VanillaJS
- HTML Web components
- CSS
- Event emitter for a basic pub/sub pattern
- Building, placing, and scaling 3D HTML components
